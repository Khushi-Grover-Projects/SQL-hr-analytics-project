# HR Analytics SQL Project

## Project Overview
Simulated HR database with 4 tables: Departments, Employees, Attendance, Payroll.  
Designed to calculate KPIs and analytics for HR decisions.

## Tables
- Departments
- Employees
- Attendance
- Payroll

## Key Analysis
- Employee Master Report
- Department-wise Headcount
- Average Salary by Department
- Attendance % per Employee
- Payroll cost and Net Salary
- Salary ranking
- KPI Dashboard

## Tools
- SQL Server Management Studio (SSMS)
- GitHub

## How to Run
1. Run 01_data.sql to create database and insert data
2. Run 02_showdata.sql to verify inserted data
3. Run 03_queries.sql to execute analysis queries
4. Check 04_output.sql for sample results
